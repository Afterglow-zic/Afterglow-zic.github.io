# Afterglow Aileron

Local Computer Use for AI agents — browser, web context, and desktop actions.
Everything runs on your own machine: the assistant you operate talks to your
browser through a loopback-only native host. No remote servers, no data
collection.

## Get started

```bash
npm install -g aileron-harness
aileron-setup
```
